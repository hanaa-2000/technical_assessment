# Flutter Technical Assessment

This project implements the provided Figma design using Flutter with a clean architecture and reusable widgets.

## Features

- Clean widget architecture
- Dynamic data models
- Responsive layout
- Staggered grid for venue selection
- Profile screen with loyalty history

## Packages Used

- flutter_staggered_grid_view
- flutter_screenutil
